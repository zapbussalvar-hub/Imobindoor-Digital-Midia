Manual ImobIndoor

Repositório: https://github.com/zapbussalvar-hub/Imobindoor
Site (GitHub Pages): https://zapbussalvar-hub.github.io/Imobindoor/

1. O que cada arquivo faz

  • index.html – tela inicial com atalhos.
  • imopai.html – painel Imopar (controle).
  • imoel.html – player Imoel (tela do elevador).
  • playlist.txt – lista de arquivos que o player vai tocar.
  • pasta media/ – onde ficam as imagens e vídeos.

2. Publicar no GitHub Pages

  1. Abra o repositório no navegador: https://github.com/zapbussalvar-hub/Imobindoor.
  2. Clique em Add file ▸ Upload files e envie os arquivos que recebeu.
  3. Depois clique em Commit changes.
  4. Vá em Settings ▸ Pages.
  5. Em Source, escolha Deploy from a branch.
  6. Em Branch, escolha main e pasta /(root), e salve.
Depois de alguns minutos, o site estará disponível em:
https://zapbussalvar-hub.github.io/Imobindoor/

3. Criar a pasta de mídias

  1. No repositório, clique em Add file ▸ Create new file.
  2. Digite o nome: media/.keep
  3. Clique em Commit changes.

Isso cria a pasta media/ onde ficarão suas imagens e vídeos.

4. Enviar imagens e vídeos

  1. Entre na pasta media/ dentro do repositório.
  2. Clique em Upload files.
  3. Escolha suas imagens (JPG, PNG) e vídeos (MP4).
  4. Clique em Commit changes para salvar.
Exemplos de nomes: banner_condominio.jpg, oferta_supermercado.png, video_lancamento.mp4

5. Editar a playlist

O arquivo playlist.txt diz ao player quais arquivos tocar, e em qual ordem.

  1. No repositório, clique em playlist.txt.
  2. Clique em Edit (ícone de lápis).
  3. Dentro dele, coloque um arquivo por linha, por exemplo:
# Comentários começam com #
banner_condominio.jpg
oferta_supermercado.png
video_lancamento.mp4
  1. Clique em Commit changes.

Pronto: o player já vai ler essa lista e tocar em loop.

6. Usar na TV Box MX9

  1. Ligue a TV Box e conecte ao Wi‑Fi.
  2. Abra o navegador (Chrome ou nativo).
  3. No endereço, digite: https://zapbussalvar-hub.github.io/Imobindoor/imoel.html
  4. Aperte Enter e espere carregar.
  5. Coloque em modo tela cheia.

Enquanto a TV estiver com essa página aberta, o elevador exibirá as mídias configuradas.

7. Onde fazer ajustes no futuro

  • Quer trocar mídias? Envie/edite arquivos na pasta media/.
  • Quer mudar a ordem? Ajuste a sequência no arquivo playlist.txt.
  • Quer ver os links principais? Abra https://zapbussalvar-hub.github.io/Imobindoor/imopai.html no computador.

Com isso você funciona como provedor de mídia indoor usando apenas GitHub + TV Box, sem servidor próprio e sem programação.
